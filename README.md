# SIMILARIDADE #

## Interface voltada para fazer análise de agrupamento ##

### Instalação ###
Para instalar:

-  baixe este repositório;
-  execute o arquivo `executar.bat`;
-  se não abrir, abra o arquivo `run.r` e execute o script, ou clique em "Run App";
-  o programa vai abrir no seu navegador padrão.

### Tutorial ###

- inserir um arquivo com extensão `.xlsx` com as observações nas linhas e variáveis nas colunas;
- informar se os dados são binário ou não;
- selecionar o algoritmo de agrupamento;
- selecionar método de agrupamento;
- clicar em `calcular`;
- um arquivo `.png` será salvo no diretório de trabalho.
